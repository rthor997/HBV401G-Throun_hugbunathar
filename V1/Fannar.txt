## User stories
1. I'm a trip planner, I'd like to see the places with the most Room's available,
so i can easily book accomidation for the large groups i oversee.

2.I'm a customer, i'd like to be able to see and sort places based on Prices,
so i can save some money for other things on my trips or go big and get a luxary room once in a while.

3. I'm a manager, i'd like to keep track of current bookings, so there is less chance of conflict of bookings.

4. I'm a customer, I'd like to be able to see how many bed's are in the room im booking,
so i can be sure the room can accomidate my family.

5. I'm a customer, i'd like to have the option of a late checkout when i book my stay, as most departures to my country are later in the day
