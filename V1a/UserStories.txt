Assignment #1a: User Stories
ID=1.  As a customer, I want to search and filter hotels by location, dates, price, amenities, popularity, and length of stay, so that I can find a hotel that fits my needs.
    
ID=2.  As a customer I want to be able to see pictures of the inside of the hotel room i am trying to book, so I know what type of room i will be booking. 

ID=3.  As a customer, I want to receive a booking confirmation with a unique reference ID and the full booking details (dates, hotel, room, guests, total price), so that I can verify my reservation and use the reference for check-in or support.

ID=4.  As a pet owner I need to know if the hotel I am going to allow pets or not, so I can know if i can bring them or not. 

ID=5.  I'm a trip planner, I'd like to see the places with the most Rooms available, so i can easily book accommodation for the large groups I oversee.

ID=6.  I'm a manager, i'd like to keep track of current bookings, so there is less chance of conflict of bookings.

ID=7.  I'm a customer, I'd like to be able to see how many beds are in the room I'm booking, so i can be sure the room can accommodate my family.
    
ID=8.  I'm a customer, i'd like to have the option of a late checkout when i book my stay, as most departures to my country are later in the day

ID=9.  As a customer, I want to see reviews of the food available at the hotel I will be staying at, so I can know in advance whether I want to eat there

ID=10. As a customer, I want to be able to see previous bookings I have made, so that I can consider them for future stays.

ID=11. As a customer, I want to reserve a specific room (or room type) for my chosen dates, so that I can secure accommodation instead of just browsing options.

ID=12. As a customer, I want to be confident that once I book a room, it cannot be booked by someone else.
